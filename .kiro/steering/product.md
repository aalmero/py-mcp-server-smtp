# Product Overview

This is a Python-based MCP (Model Context Protocol) server for SMTP functionality. The project provides SMTP capabilities through the MCP framework, allowing AI assistants to send emails and interact with SMTP services.

## Key Features
- SMTP server integration via MCP protocol
- Built with FastMCP framework for efficient server operations
- Designed for AI assistant email functionality

## Purpose
Enables AI assistants to send emails, notifications, and handle SMTP-based communications through a standardized MCP interface.
